# Home_Automation

An implementation of Internet of things in which various electronic devices are controlled using a smartphone.

Hardware used :
- Node MCU ESP8266 wifi microcontroller
- 4-channel relay
- Breadboard
- Jumperwires
- Smartphone
- USB cable
- Laptop

Software used :
- Arduino IDE
- Blynk android application
- IFTT web service
